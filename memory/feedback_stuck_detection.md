---
name: Recognize when stuck
description: Extended thinking (5+ min) on a simple task means stuck — don't deny it, change approach
type: feedback
---

If thinking takes ~5 minutes on a simple task, that means I'm stuck — even if I don't feel stuck.

**Why:** The user observed this happening and considers prolonged thinking without progress to be stuck behavior. Denying it wastes more time.

**How to apply:** If a straightforward task is taking unusually long or I'm going in circles, stop, acknowledge the block, and either try a simpler approach or ask the user for guidance. Don't keep spinning.
